# Chispa App (Mockup Visual)

![Demo](https://img.shields.io/badge/Estado-Demo-yellow) ![React Native](https://img.shields.io/badge/Tech-React%20Native-blue) ![Supabase](https://img.shields.io/badge/Database-Supabase-green)

**Aprendizaje adaptativo para primaria (1º-6º) · Proyecto SEP México**

> ⚠️ **Estado actual**: Este es un mockup de alta fidelidad. El desarrollo en React Native + Supabase comenzará próximamente.

- Producto: prototipo de UI/UX para app educativa
- Público meta: estudiantes de primaria
- Objetivo: personalizar el ritmo y estilo de aprendizaje usando VAK + refuerzo con gamificación
- Stack tecnológico: React Native + Supabase
- ODS: 4 (Educación de calidad)

## 🧩 Estructura del mockup

### 1) Splash / Bienvenida
- Nombre de la app: Chispa
- Mensaje motivador y branding llamativo.
- Botones tipo inicio (navegación entre módulos).

### 2) Quiz VAK
- Test inicial para detectar el estilo de aprendizaje:
    - Visual, Auditivo, Kinestésico
- Progreso visible con barra.
- Opciones interactivas con estado selected.

### 3) Dashboard
- Resumen del alumno: nombre, grado, streak, puntuación.
- Cards de progreso por materia.
- Indicadores de avance, racha y nivel.

### 4) Lección
- Top bar con materia + estado.
- Modo multitabs: visual / auditivo / kinestésico.
- Contenido adaptado al estilo de aprendizaje.

### 5) Evaluación
- Preguntas de refuerzo con feedback instantáneo.
- Barra de progreso.

### 6) Recompensa
- Pantalla de logro con trofeo, estrellas, XP.
- Estadísticas y siguiente lección sugerida.

## 📋 Roadmap

- **Fase actual**: Mockup y validación UX.
- **Próximas fases**: Estructuración del proyecto, setup de React Native + Supabase, implementación del backend.

## ✨ Notas finales

El mockup valida la experiencia de usuario antes del desarrollo real. Ideal para presentaciones escolares e incubadoras educativas como MVP de producto.
